
# Calculator

Basic Calculator application using react js 


## Run Locally

Clone the project

```bash
  git clone https://github.com/Parimal2607/calCulator
```

Go to the project directory

```bash
  cd maccal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
